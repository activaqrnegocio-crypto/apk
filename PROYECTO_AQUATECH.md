# Aquatech CRM - Documentación del Proyecto

## Descripción
Sistema CRM para Aquatech con funcionalidades offline y sincronización.

## Estructura del Proyecto
- Frontend: Next.js con PWA
- Backend: API con Next.js
- Base de datos: PostgreSQL con Prisma

## Características Principales
- Proyectos y cotizaciones
- Inventario de materiales
- Calendario de actividades
- Sistema de notificaciones
- Offline support con Service Worker

## API Endpoints
- `/api/projects` - Gestión de proyectos
- `/api/clients` - Gestión de clientes
- `/api/materials` - Inventario
- `/api/quotes` - Cotizaciones
- `/api/users` - Usuarios del sistema

## Scripts Disponibles
- `npm run dev` - Desarrollo
- `npm run build` - Build de producción
- `npm run start` - Servidor de producción
